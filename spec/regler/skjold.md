# Skjold

## Oppstilling

Hver spiller har ett skjold. Hvitt skjold starter på felt l1, svart skjold starter på felt l12.

## Flytteregler

- Skjoldet kan flytte flere felter om gangen, og kan hoppe over andre brikker.
- Den kan flytte horisontalt og vertikalt.
- Skjoldet kan ikke gå til et felt hvor det står en brikke med samme farge.

## Å ta motstanders brikker

- Skjoldet kan ta alle brikker som står der den kan flytte.
